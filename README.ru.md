# Fraction Conquest

Fraction Conquest — браузерный учебный тренажер по дробям для 5-7 классов (11-13 лет).  
Игрок решает математические задания и захватывает территории на карте: правильные ответы расширяют контроль, ошибки приводят к потерям.

Языковые версии:

- Русский: `README.ru.md`
- Немецкий: `README.de.md`

## Идея приложения

Проект объединяет:

- Практику по дробям и пошаговое понимание решения
- Короткие игровые циклы с наглядным прогрессом (захват карты)
- Настраиваемую сложность и темы для школы и самостоятельного обучения

Цель — сделать регулярную практику дробей более вовлекающей, без внешних зависимостей и сложного разворачивания.

## Основные возможности

- 11 тем по дробям (сокращение, смешанные числа, операции, десятичные, комбинированные выражения)
- 4 уровня сложности
- Игровая механика захвата территорий с мгновенной обратной связью
- Пояснения при неверном ответе
- Локализация интерфейса RU/DE
- Страница администрирования настроек (`admin.html`)
- Встроенная страница тестов математического движка (`test.html`)
- Чистый фронтенд: HTML + CSS + JavaScript (без фреймворков и сборки)

## Как запустить на своем компьютере

### Вариант 1: Открыть напрямую (самый быстрый)

1. Скачайте репозиторий:
   - `git clone https://github.com/gluonMaster/Educational-game.git`
   - или скачайте ZIP с GitHub и распакуйте
2. Откройте папку проекта.
3. Запустите файлы в браузере:
   - Игра: `index.html`
   - Настройки: `admin.html`
   - Тесты: `test.html`

### Вариант 2: Через локальный сервер (рекомендуется)

В папке проекта выполните:

```bash
python -m http.server 8000
```

Затем откройте:

- `http://localhost:8000/index.html`
- `http://localhost:8000/admin.html`
- `http://localhost:8000/test.html`

## Управление

- Мышь: клик по одному из 6 вариантов ответа
- Клавиатура: клавиши `1-6` для выбора ответа
- После неверного ответа переход к следующей задаче по кнопке `Продолжить`

## Технические примечания

- Рекомендуемое разрешение: `1024x768` и выше
- Целевые браузеры: Chrome/Firefox/Edge 90+
- Настройки и прогресс сохраняются в `localStorage`
